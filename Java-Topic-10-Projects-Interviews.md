## üî∑ Part 4: Interview Preparation

### Common Interview Questions

**Core Java:**
1. Explain OOP concepts with examples
2. Difference between ArrayList and LinkedList
3. How does HashMap work internally?
4. What is multithreading? Explain with example
5. Exception handling best practices

**Spring Boot:**
1. What is dependency injection?
2. Explain Spring Boot auto-configuration
3. How to create REST API in Spring Boot?
4. What is @Autowired annotation?
5. Difference between @Component, @Service, @Repository

**Database:**
1. Explain normalization with example
2. What are indexes? When to use?
3. Difference between INNER JOIN and LEFT JOIN
4. How to optimize slow queries?
5. Explain ACID properties

**System Design:**
1. Design a URL shortener
2. Design a notification system
3. How would you scale a web application?
4. Explain microservices architecture
5. Caching strategies

---

### Behavioral Questions (STAR Method)

**Situation ‚Üí Task ‚Üí Action ‚Üí Result**

**Example:**
Q: "Tell me about a challenging project"

A: 
- **Situation:** "In my e-commerce project, I faced performance issues with 1000+ concurrent users"
- **Task:** "I needed to reduce API response time from 2s to under 500ms"
- **Action:** "I implemented Redis caching for frequently accessed data and optimized database queries with proper indexing"
- **Result:** "Response time reduced to 300ms, improving user experience by 85%"
